# Imagen de Fondo

Para usar tu propia imagen de fondo (la imagen "Wonder Alien" que adjuntaste):

1. Guarda tu imagen en esta carpeta con el nombre `fondo.jpg` o `fondo.png`
2. La imagen se aplicará automáticamente como fondo de la página inicial

## Formatos soportados
- JPG/JPEG
- PNG
- WebP

## Recomendaciones
- Resolución mínima recomendada: 1920x1080 píxeles
- Tamaño de archivo: menor a 2MB para mejor rendimiento
- Formato preferido: JPG para fotografías, PNG para ilustraciones

La aplicación ya viene con un fondo por defecto (un paisaje estilo Mario Bros con cielo azul y plataformas verdes). Si quieres usar tu propia imagen, simplemente colócala aquí con el nombre `fondo.jpg`.
